# SpacemiT Open Source Upstream Summary

Welcome to the SpacemiT Open Source Upstream Summary page!

This page summarizes the upstream progress of SpacemiT K1/K3 support across open-source projects, including system software, toolchains, AI software, and SpacemiT-maintained independent open-source projects.

It provides:
- A single overview table linking to each project's detailed status page
- A unified view of K1/K3 support progress across open-source software
